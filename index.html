What is numpy ?
Numpy is a package of python which is the major one used for machine learning. Which help us to build array, a single dimension or a multi-dimensional arrays.We can also operate these arrays with a large amount of data. In 2005, Travis Oliphant created NumPy by incorporating features of the competing Numarray into Numeric, with extensive modifications. Numpy is an open-source software and it has many contributors.
What is pandas ?
Pandas is a python library used for working with datasets. Which helps us to clean , manipulate and analyze data easily. The name Pandas has a reference with both "Panel Data" and "Python Data Analysis". Development of pandas started in 2008, pandas becomes open source in 2009 and First edition of Python for Data Analysis is published in 2012. Pandas was created by Wes McKinney in the year 2007 to 2010. He is an american software developer and a businessman.
What is Matplotlib ?
So in the above paragraph you see about pandas, which helps in working with datasets. Now Matplotlib will helps us to visualize the data from the dataset so we can easily look at the building of the dataset. We can use bar-chart, histogram, and also line-chart with help of matplotlib. Matplotlib was originally written by John D. Hunter. Since then it has an active development community and is distributed under a BSD-style license. Michael Droettboom was nominated as matplotlib's lead developer shortly before John Hunter's death in August 2012 and was further joined by Thomas Caswell.
